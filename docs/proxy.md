## 我为什么要使用代理？

ip代理可以理解为伪装你的ip。

由于反爬虫最重要的手段之一就是通过ip判断行为，如果一个ip快速频繁访问，则判定这个ip为爬虫。

因此通过不断的变换你的ip来迷惑服务器。

## 代理的效果怎么样？

目前的版本，由于已经适配了加密接口（非登录数据），因此可以在代理的加持下可以大规模爬取数据。

但是对于登录数据，代理的加持并不那么明显（但是从我几次试验的效果来看，依然有一定效果）。

## ip代理说明

ip代理，目前只适配了json格式，还没有适配秘钥格式的代理。

json格式由于各家代理json格式也不一样，目前只适配了一种格式：
    
    [
        {
            "ip":"xxx.xxx.xxx.xxx",
            "port":"xxxxx"
        },
        {
            "ip":"xxx.xxx.xxx.xxx",
            "port":"xxxxx"
        },
    ]
    
已知还有另一种json格式，并没有适配

    "state":"ok",
    "data":[
            {
                "ip":"xxx.xxx.xxx.xxx",
                "port":"xxxxx"
            },
            {
                "ip":"xxx.xxx.xxx.xxx",
                "port":"xxxxx"
            },
        ]
    
但是本质上没区别，如果需要自行更改。
